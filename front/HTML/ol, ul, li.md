# ol
- `ol`(Ordered List)태그는 이가 가지는 항목이 순서와 유관한 경우 사용하는 목록 태그이다.
- 자동으로 닫는 태그 : X
- 자식으로 li 태그를 가진다.
# ul
- `ul`(Unordered List)태그는 이가 가지는 항목이 순서와 무관한 경우 사용하는 목록 태그이다.
- 자동으로 닫는 태그 : X
- 자식으로 li 태그를 가진다.
- 주로 메뉴를 만들 때 사용한다.
# li
- `li`(List item)태그는 목록의 항목 하나를 의미하는 태그이다.
- 자동으로 닫는 태그 : X
- 부모로 ol 혹은 ul 태그를 가진다.
